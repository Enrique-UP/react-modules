Object Methods

https://www.w3schools.com/jsref/jsref_obj_object.asp
  1. constructor Returns the function that created an object's prototype
  2. keys() Returns an Array Iterator object with the keys of an object
  3. prototype Let you to add properties and methods to JavaScript objects
  4. toString() Converts an object to a string and returns the result
  5. valueOf() Returns the primitive value of an object
