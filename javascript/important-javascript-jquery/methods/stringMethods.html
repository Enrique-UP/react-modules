String Methods

  1. length Returns the length of a string
  2. slice() Extracts a part of a string and returns a new string
  3. substring() Extracts characters from a string, between two specified indices (positions)
  4. substr() Extracts a number of characters from a string, from a start index (position)
  5. replace() Searches a string for a value, or a regular expression, and returns a string where the values are replaced
  6. String replaceAll()
  7. toUpperCase() Returns a string converted to uppercase letters
  8. toLowerCase() Returns a string converted to lowercase letters
  9. concat() Returns two or more joined strings
  10. trim() Returns a string with removed whitespaces
  11. trimEnd() Returns a string with removed whitespaces from the end
  12. trimStart() Returns a string with removed whitespaces from the start
  13. String padStart()
  14. String padEnd()
  15. charAt() Returns the character at a specified index (position)
  16. charCodeAt() Returns the Unicode of the character at a specified index
  17. split() Splits a string into an array of substrings
  18. constructor Returns the string's constructor function
  19. endsWith() Returns if a string ends with a specified value
  20. fromCharCode() Returns Unicode values as characters
  21. includes() Returns if a string contains a specified value
  22. indexOf() Returns the index (position) of the first occurrence of a value in a string
  23. lastIndexOf() Returns the index (position) of the last occurrence of a value in a string
  24. localeCompare() Compares two strings in the current locale
  25. match() Searches a string for a value, or a regular expression, and returns the matches
  26. prototype Allows you to add properties and methods to an object
  27. repeat() Returns a new string with a number of copies of a string
  28. search() Searches a string for a value, or regular expression, and returns the index (position) of the match
  29. startsWith() Checks whether a string begins with specified characters
  30. toLocaleLowerCase() Returns a string converted to lowercase letters, using the host's locale
  31. toLocaleUpperCase() Returns a string converted to uppercase letters, using the host's locale
  32. toString() Returns a string or a string object as a string
  33. valueOf() Returns the primitive value of a string or a string object
