Array Methods

  1. toString() Converts an array to a string, and returns the result
  2. join() Joins all elements of an array into a string
  3. pop() Removes the last element of an array, and returns that element
  4. push() Adds new elements to the end of an array, and returns the new length
  5. shift() Removes the first element of an array, and returns that element
  6. unshift() Adds new elements to the beginning of an array, and returns the new length
  7. length Sets or returns the number of elements in an array
  8. delete
  9. flat
  10. splice() Adds/Removes elements from an array
  11. slice() Selects a part of an array, and returns the new array
  12. sort() Sorts the elements of an array
  13. reverse() Reverses the order of the elements in an array
  14. forEach() Calls a function for each array element
  15. map() Creates a new array with the result of calling a function for each array element
  16. filter() Creates a new array with every element in an array that pass a test
  17. concat() Joins arrays and returns an array with the joined arrays
  18. constructor Returns the function that created the Array object's prototype
  19. copyWithin() Copies array elements within the array, to and from specified positions
  20. entries() Returns a key/value pair Array Iteration Object
  21. every() Checks if every element in an array pass a test
  22. fill() Fill the elements in an array with a static value
  23. find() Returns the value of the first element in an array that pass a test
  24. findIndex() Returns the index of the first element in an array that pass a test
  25. from() Creates an array from an object
  26. includes() Check if an array contains the specified element
  27. indexOf() Search the array for an element and returns its position
  28. isArray() Checks whether an object is an array
  29. keys() Returns a Array Iteration Object, containing the keys of the original array
  30. lastIndexOf() Search the array for an element, starting at the end, and returns its position
  31. prototype Allows you to add properties and methods to an Array object
  32. reduce() Reduce the values of an array to a single value (going left-to-right)
  33. reduceRight() Reduce the values of an array to a single value (going right-to-left)
  34. some() Checks if any of the elements in an array pass a test
  35. valueOf() Returns the primitive value of an array
