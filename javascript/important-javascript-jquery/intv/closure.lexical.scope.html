A closure is a function having access to the parent scope, even after the parent function has closed.
        क्लोजर एक ऐसा फ़ंक्शन है जिसके पास पैरेंट फ़ंक्शन बंद होने के बाद भी पैरेंट स्कोप तक पहुंच होती है।
        Or
        In other words, closure is created when a child function keep the environment of the parent scope even after the parent function has already executed 
        दूसरे शब्दों में, क्लोजर तब बनता है जब एक चाइल्ड फ़ंक्शन पैरेंट फ़ंक्शन के पहले ही निष्पादित होने के बाद भी पैरेंट स्कोप के वातावरण को बनाए रखता है
        
        
        // Example 1
        var sum = function(a){
            console.log("Live Viewers " + a);
            var c = 4;
            return function(b){
                return a + b + c;
            }
        }
        var store = sum(2);
        console.log(store(5));

        // Example 2
        var sumAll = function(a,b,c){
            return {
                sumTwo:function(){
                    return a + b;
                },
                sumThree:function(){
                    return a + b + c;
                }
            }
        }
        var store = sumAll(2,10,5);
        console.log(store.sumTwo());
        console.log(store.sumThree());

        var store1 = sumAll(7,8,9);
        console.log(store1.sumTwo());
        console.log(store1.sumThree());
